## 📱 Mobile Pe Install Karne Ke Steps

### Step 1: GitHub Account Banayein (Free)
1. **github.com** kholen browser mein
2. "Sign Up" click karein
3. Account banayein (free hai)

---

### Step 2: New Repository Banayein
1. GitHub login karne ke baad, **"+" icon** click karein (top right)
2. **"New repository"** select karein
3. Repository name: `ramgarh-digital-lens`
4. **Public** select karein
5. **"Create repository"** click karein

---

### Step 3: Files Upload Karein
1. Repository page pe **"uploading an existing file"** link click karein
2. Is ZIP ke andar ke **saare folders aur files** drag karein:
   - `src/` folder
   - `public/` folder
   - `package.json`
   - `vercel.json`
3. **"Commit changes"** click karein

---

### Step 4: Vercel Pe Deploy Karein (Free)
1. **vercel.com** kholen
2. "Sign Up" → **"Continue with GitHub"** choose karein
3. GitHub se login karein
4. **"Add New Project"** click karein
5. `ramgarh-digital-lens` repository select karein
6. **"Deploy"** click karein — bas! 🚀

---

### Step 5: Mobile Pe Install Karein
1. Vercel deployment complete hone ke baad **URL milega** (e.g. `ramgarh-lens.vercel.app`)
2. **Android Chrome** mein URL kholen
3. Top right **3 dots menu** click karein
4. **"Add to Home Screen"** click karein
5. **"Add"** confirm karein

✅ Ab Ramgarh Digital Lens aapke mobile pe app ki tarah install ho gaya!
